== Description ==

The **Emargy Elements** plugin is a custom addon for **Elementor** that allows you to create an interactive timeline to showcase **posts, projects, services, or any other content** in a professional way.

### Key Features:
- **Fully compatible with Elementor**, allowing easy drag-and-drop usage.
- **Dynamic content display** from any WordPress post type.
- **Fully responsive design** that adapts to all devices.
- **Interactive timeline layout** inspired by a soundwave.
- **Highlighted center post**, displayed at twice the size for better emphasis.
- **Smooth navigation** between posts.
- **Direct post opening** when clicking on an item.
- **Future-proof expansion**, supporting Arabic language and fully compatible with the latest versions of **WordPress** and **Elementor**.

### Timeline Showcase Widget

The main widget included in this plugin allows you to create beautiful and interactive timelines for your content. It comes with multiple display options and styling capabilities to match your site's design perfectly.

== Installation ==

1. Download the plugin ZIP file.
2. Go to **WordPress Dashboard > Plugins > Add New**.
3. Click **Upload Plugin** and select the file.
4. After installation, click **Activate**.

== Usage Instructions ==

1. Add the widget to your page using Elementor.
2. Choose the **content type** you want to display.
3. Adjust **layout and design settings** according to your preference.
4. Save the page and enjoy the professional timeline display.

== Shortcodes ==

If you prefer using the widget outside of Elementor, use the following shortcode:
`[emargy_timeline type="portfolio" limit="10" layout="wave" featured_size="2x"]`

**Parameters:**
- `type`: Content type (`portfolio`, `post`, `service`, etc.).
- `limit`: Number of items displayed.
- `layout`: Timeline design (`wave`, `straight`, `custom`).
- `featured_size`: Center post size ratio.

== Frequently Asked Questions ==

= Can I use this plugin with custom content? =
Yes, you can select any custom post type in the widget settings.

= Does the timeline update dynamically? =
Yes, the content updates automatically when new posts are added.

= Can I change the colors and design? =
Absolutely! You can fully customize colors, fonts, and backgrounds from the widget settings.

= Can I navigate the timeline by dragging or clicking arrows? =
Yes, you can enable drag navigation or use navigation arrows.
